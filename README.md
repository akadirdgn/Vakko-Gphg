# Vakko Event UI — Uzman CRM Staj Projesi

**Kısa açıklama:** Vakko için hazırlanmış etkinlik (event) landing page UI'si. Uzman CRM stajı sırasında tamamen bana ait olarak tasarlandı ve uygulandı. Proje şu anda canlı: [https://event.vakko.com.tr/](https://event.vakko.com.tr/).

## Hakkında
Bu depo, Vakko etkinlik sayfasının (event landing page) kullanıcı arayüzü (UI) uygulamasını içerir. Tüm tasarım ve ön yüz geliştirmesi tarafımdan (akadirdgn) yapılmıştır. Proje staj sürecinde geliştirildi ve şu an canlı olarak erişilebilir.

Canlı site: [https://event.vakko.com.tr/](https://event.vakko.com.tr/) — (Canlıda şu an)

## Öne çıkan özellikler
- Duyarlı (responsive) tasarım — masaüstü, tablet ve mobil uyumlu

## Lokal Çalıştırma (örnek)
Aşağıdaki adımlar projeye göre değişebilir. Depoda bir package.json yoksa doğrudan `index.html` dosyasını tarayıcıda açabilirsiniz.

1. Depoyu klonlayın:
   git clone https://github.com/akadirdgn/Vakko-Gphg.git
2. Klasöre girin:
   cd Vakko-Gphg
3. (Varsa) Bağımlılıkları yükleyin:
   npm install
4. (Varsa) Geliştirme sunucusunu çalıştırın:
   npm run dev
5. Alternatif (statik): `index.html` dosyasını tarayıcıda açın veya basit bir HTTP sunucusu kullanın:
   npx serve .

Eğer proje farklı bir yapı/komut kullanıyorsa bana söyleyin; README'yi tam komutlarla güncelleyeyim.
